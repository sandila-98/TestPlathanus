Instruções para execução:

- Camada visual foi ultizido o React.
   Para executar é preciso entrar na pasta "./TestPlathanus/plathanus" através do Prompt de Comando ou Terminal Linux e executar o código "npm start".
   O projeto abrirá na URL "localhost:3000".

-  Para o back-end utilizamos o Nodejs e para a criação do servidor instalamos o Express.
   Para rodar o servidor entre na pasta "./TestPlathanus/servidor" e executar o código "npm start" através do Prompt de Comando ou TerminaL Linux.
   Abrir o navegador e digitar na URL "localhost:3001".
